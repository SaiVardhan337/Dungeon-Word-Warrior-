# 📂 Dungeon Typer — Examples & Snippets

This folder provides standalone code snippets and integration examples showing how to customize or reuse components from **Dungeon Typer**.

---

## 📄 Available Examples

### 1. `custom-wordlist.js`
*   **Description**: Demonstrating how to override or inject custom vocabulary tiers into the game engine (e.g. Developer terms, Medical terms, or Foreign languages).
*   **Usage**: Include the script after `game.js` in `index.html` or invoke `applyDeveloperVocabulary()` dynamically.

### 2. `audio-synth-standalone.html`
*   **Description**: A single-file HTML demonstration showing how the zero-dependency Web Audio synthesizer works independently without external sound files.
*   **Usage**: Open `audio-synth-standalone.html` in any web browser and click the buttons to test procedural audio synthesis!
